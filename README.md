# BuildHeightRestrictor

建造高度限制 `1.12.2及以上`  
额外支持`folia` `CatServer` `Arclight`

> 拥有 `buildheightrestrictor.bypass` 可以绕过限制  
> `/bhr reload 重载配置文件`

```yaml
# 配置文件版本
ver: 1

# 限制建造是的消息提醒
Message:
  - "你不能在这里建造"

# 限制的高度(256的意思是256还可以放置，高于就不行了)
Height: 256
```
